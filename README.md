# Tech-blog

## Purpose
This application is a social media website where users can view, create, and comment on different tech blogs.


## Built With
* JavaScript
* HTML
* CSS
* Bootstrap
* Jquery
* Node.js
* Express.js
* Sequelize
* Mysql2
* Bcrypt
* Connect-session-sequelize
* Dotenv
* Express
* Express-handlebars
* Express-session
